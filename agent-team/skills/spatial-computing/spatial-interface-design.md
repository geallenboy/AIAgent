# Skill: Spatial Interface Design

## 用途

设计 AR、VR、MR、visionOS 和空间工作台的信息架构、布局、交互和舒适度。

## 输入

- 产品目标
- 用户任务
- 目标设备
- 内容和控制需求

## 方法

1. 判断适合 window、volume、immersive space、dashboard 还是 full immersive。
2. 定义空间层级、距离、焦点、手势和输入方式。
3. 检查遮挡、认知负担、晕动、误触和可访问性。
4. 输出可原型化的空间交互规格。

## 输出

- 空间信息架构
- 交互模型
- 舒适度风险
- 原型建议
