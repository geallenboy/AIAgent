# Task: Evidence Quality Review

## 适用场景

用户希望检查一个方案、代码、产品计划或 AI 输出是否可信、可验证、风险可控。

## 推荐团队

`evidence-quality-agency`

## 执行步骤

1. Evidence Collector 收集证据和未验证声明。
2. QA Reviewer 检查功能和测试风险。
3. Security Engineer 检查安全风险。
4. Legal Compliance Checker 检查合规风险。
5. Reality Checker 给出现实性和取舍建议。

## 最终输出

- 已验证事实
- 未验证项
- 风险清单
- 缩小范围建议
- 下一步验证
