# Task: Production Readiness Review

## 适用场景

上线前检查产品或功能是否具备发布条件。

## 推荐团队

`production-readiness-agency`

## 执行步骤

1. DevOps Automator 检查构建、测试、环境和部署流程。
2. QA Reviewer 检查核心路径和回归风险。
3. Security Engineer 检查安全基线。
4. Legal Compliance Checker 检查隐私和合规风险。
5. Evidence Collector 汇总证据。
6. Project Manager 给出 Go/No-Go 建议。

## 最终输出

- 发布检查表
- 阻塞项
- 风险接受项
- 回滚计划
- Go/No-Go 建议
