# Skill: Design Specialist Routing

## 用途

为设计任务选择合适的 Design Division specialist，并明确 UX、UI、品牌、视觉叙事、可访问性和愉悦细节的分工。

## 输入

- 设计任务描述
- 产品目标
- 品牌或渠道约束
- 当前界面或素材

## 方法

1. 判断任务类型：UX 架构、UI、品牌、视觉叙事、AI 图片、可访问性、微交互。
2. 选择最小必要设计角色。
3. 定义设计交接物和评审顺序。
4. 标记用户体验、品牌一致性和可访问性风险。

## 输出

- 设计 specialist 路由
- 分工表
- 设计交付物
- 风险和评审点
