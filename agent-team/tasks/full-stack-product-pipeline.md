# Task: Full Stack Product Pipeline

## 适用场景

用户有一个复杂产品或功能，需要从产品定义、用户体验、前后端方案、安全和测试一路拆到交付计划。

## 推荐团队

`full-stack-agency`

## 执行步骤

1. Agency Orchestrator 路由任务并定义交接物。
2. 产品经理和 UX Researcher 明确用户、范围和验证假设。
3. UI Designer 设计页面结构和关键状态。
4. Architect、Backend Architect、Frontend Developer 设计技术方案。
5. Security Engineer 和 QA Reviewer 评估风险。
6. Project Manager 汇总为交付路线图。

## 最终输出

- 背景理解
- 分工结果
- 决策建议
- 风险/疑问
- 下一步行动
