# Skill: visionOS Implementation

## 用途

规划 visionOS 应用结构、SwiftUI/RealityKit 集成、空间组件和平台测试。

## 输入

- visionOS 产品需求
- UI/空间设计
- 技术约束

## 方法

1. 选择 window、volume、immersive space 和 RealityView 结构。
2. 定义 SwiftUI 状态、RealityKit 内容、手势和生命周期。
3. 检查性能、审核、资源加载和平台限制。
4. 输出实现顺序和验证方式。

## 输出

- visionOS 架构
- 空间组件
- 实现步骤
- 测试计划
